NBA Analytics

Welcome to our CS180 project for XYZ Coders. In this project we worked with NBA data that included the seasons 2004 - 2020.

Authors: Edward Zhu, Nikhil Kadekar, Arya Faramarzi, Jorge Peng, Aramis Masarati

Select Query

Player list: Given a number of players, the output is the number of NBA players in table format from 2004 - 2020.

Conference: Given an option to select West or East, the output are all the teams in either East or West Conference.

Team Wins: Given an option to select seasons 2004 - 2020, the output are all the teams that played in that season and their amount of wins.

Home Wins: Given an option to select seasons 2004 - 2020, the output are all the teams that played in that season and their home record.

Manipulate Data

This section allows the user to Insert, Modify, or Update data. The modifiable data are the players, teams, ranking.

The players modifiable data include PlayerID, TeamID, Player Name, and Season Played.

The teams modifiable data include Team Name, Team Abbreviation, Year Founded, and City.

The ranking modifiable data include Team, TeamID, Season, Wins, Losses, Home Record, Away Record.

Compute Analytics

This section computes the data and output an analytic. The following available analytic are the following:

Total Player Stats: Given a player name, the output gives the number of Points, Rebounds, Assists, Steals, and Blocks

Largest Point Difference In A Game: Given a season, the outputs is the number of points of two teams and the difference that had the highest point difference.

Average Stats Of A Winning Team: Given a season, the output is the Avg. Field Goal Percentage, Avg. 3PT Percentage, Avg. Free Throw Percentage, Avg. Points Per Game, Avg. Rebounds Per Game, Avg. Assists Per Game, for winning teams.

Head To Head Record Between Two Teams: Given a season and two teams, the output is their match record between each other.

Average Home vs. Away Stats: Given a season, the output is the Avg. Free Throw Percentage and Avg. 3PT Percetange and compares the Home team and Away Team.

Team With The Most Home And Away Teams Wins: Given a season, the output is the team and the number of wins for the highest Home record and highest Away Record.

Incremental Analytics

The following analytics use an incremental implementation. After the first computation, followed by an Insert, Update, or Delete, the next computations are computed in a way faster manner by saving the data beforehand.
